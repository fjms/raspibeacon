# Raspibeacon

Escaneo de beacon ble Kontack enviando una peticion post a un endpoint



```js
npm install raspibeacon
var raspibeacon = require('raspibeacon')
raspibeacon.startScan("http://localhost:80", "endpoint/beacon");
```
